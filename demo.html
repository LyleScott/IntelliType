I wrote this to be WSGI compliant so you can use it any webserver that can support WSGI. For this demo, just run the intellitype.wsgi Python program directly to launch a small HTTP server provided by the WSGI module that is in the Python standard library. See the full source at https://github.com/LyleScott/IntelliType.

  • Make sure the lxml Python library is installed.
  • Start the server by running intellitype.wsgi with python.
    Note: listens on port 8080, but you can specify another port as an argument.

As you type, the program suggest autocompletion options based on what has previously been entered.

  1. the cat and the fox<ENTER>
  2. the cat and the mouse<ENTER>
  3. the cat and the mouseymouse<ENTER>
  4. the cat and the mou<WAIT-AND-LOOK>

You can browse the xmls directory to view the generated XML. New entries are inserted into the XML in a tree-like fashion for easy searching. When a search query comes in, the entire query is split into tokens (think words). The suggestions are the child nodes of the full path represented by the current series of tokens, minus the last one (though, if the last token is being typed, child nodes are filtered such that all strings start with the token that is being typed).

Try it out: type to search or press enter to add the text input to the tree of possible suggestions.
