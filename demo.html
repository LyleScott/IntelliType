I wrote this to be WSGI compliant to you can use it with most webservers. For this demo, just run the intellitype.wsgi Python program directly to launch a small HTTP server provided by the WSGI module that is in the Python standard library.

  • Make sure the lxml Python library is installed.
  • Start the server by running intellitype.wsgi with python.

As you type, the program suggest autocompletion options based on what has previously been entered.

  1. the cat and the fox<ENTER>
  2. the cat and the mouse<ENTER>
  3. the cat and the<WAIT-AND-LOOK>

You can browse the xmls directory to view generated XML.
